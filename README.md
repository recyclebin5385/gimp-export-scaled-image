# GIMP Export Scaled Image Plug-in

## Summary

A plug-in for GIMP 3 that exports images at a different size from the original.

## Installation

Place the entire folder into GIMP's plug-in directory.

The folder name must match the script name:
`export-scaled-image`

## Usage

Select **File -> Export Scaled Image...** from the main menu to open the dialog.  
Set the desired parameters and click **OK** to export the image.
